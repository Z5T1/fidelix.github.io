INSTALLPKG(8) Linux System Administrator's Manual INSTALLPKG(8)

NAME

installpkg - install a package

SYNOPSIS

installpkg [-hvr] [-d ROOT_DIR] <PACKAGE_FILE>

DESCRIPTION

Install a PACKAGE_FILE that was created with makepkg(8)

Options:

-d <root directory>
Causes the package to be installed to the specified root directory instead of the default of /.
-r
If the package is already installed, reinstall it.
-h
Displays a help message.
-v
Print version information.
