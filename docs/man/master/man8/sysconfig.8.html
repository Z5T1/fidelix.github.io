SYSCONFIG(8) Linux System Administrator's Manual SYSCONFIG(8)

NAME

sysconfig - An interactive tool for configuring the system

SYNOPSIS

sysconfig [SUBSYSTEM]

DESCRIPTION

An interactive tool for configuring the system. Subsystems are listed in /etc/sysconfig.d. If no subsystem is specified, sysconfig will display a menu of subsystems for selection.

AUTHOR

Sysconfig was written by Scott Court (z5t1@z5t1.com).

HISTORY

Sysconfig first appeared in Fidelix 0.1
