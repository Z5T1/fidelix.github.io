REMOVEPKG(8) Linux System Administrator's Manual REMOVEPKG(8)

NAME

removepkg - remove a package

SYNOPSIS

removepkg [-h] [-d ROOT_DIR] <PACKAGE_NAME>

DESCRIPTION

Uninstalls the package PACKAGE_NAME that was previously installed with installpkg(8) This will remove all of the files provided by the package, except for files that are also provided by one or more other packages. It also removes any directories provided by the package that are empty after the package is uninstalled.

Options:

-d <root directory>
Causes the package to be removed from the specified root directory instead of the default of /.
-h
Displays a help message.
