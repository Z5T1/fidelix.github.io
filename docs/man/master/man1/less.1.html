LESS(1) User Commands LESS(1)

NAME

less - manual page for less 1.31.1

SYNOPSIS

less [-EFIMmNSRh~] [FILE]...

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

View FILE (or stdin) one screenful at a time

-E
Quit once the end of a file is reached
-F
Quit if entire file fits on first screen
-I
Ignore case in all searches
-M,-m
Display status line with line numbers and percentage through the file
-N
Prefix line number to each line
-S
Truncate long lines
-R
Remove color escape codes in input
-~
Suppress ~s displayed past EOF
