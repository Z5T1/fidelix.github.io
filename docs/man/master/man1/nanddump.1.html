NANDDUMP(1) User Commands NANDDUMP(1)

NAME

nanddump - manual page for nanddump 1.31.1

SYNOPSIS

nanddump [-no] [--bb padbad|skipbad] [-s ADDR] [-l LEN] [-f FILE] MTD_DEVICE

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Dump MTD_DEVICE

-n
Read without ecc
-o
Dump oob data

-s ADDR Start address

-l LEN
Length

-f FILE Dump to file ('-' for stdout)

--bb METHOD

skipbad: skip bad blocks padbad: substitute bad blocks by 0xff (default)
