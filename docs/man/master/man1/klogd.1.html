NAME

klogd - manual page for klogd 1.31.1

SYNOPSIS

klogd [-c N] [-n]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Log kernel messages to syslog

-c N
Print to console messages more urgent than prio N (1-8)
-n
Run in foreground
