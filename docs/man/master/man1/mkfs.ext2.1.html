MKFS.EXT2(1) User Commands MKFS.EXT2(1)

NAME

mkfs.ext2 - manual page for mkfs.ext2 1.31.1

SYNOPSIS

mkfs.ext2 [-Fn] [-b BLK_SIZE] [-i INODE_RATIO] [-I INODE_SIZE] [-m RESERVED_PERCENT] [-L LABEL] BLOCKDEV [KBYTES]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.
-b BLK_SIZE
Block size, bytes
-F
Force
-i RATIO
Max number of files is filesystem_size / RATIO
-I BYTES
Inode size (min 128)
-L LBL
Volume label
-m PERCENT
Percent of blocks to reserve for admin
-n
Dry run
