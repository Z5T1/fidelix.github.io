BRCTL(1) User Commands BRCTL(1)

NAME

brctl - manual page for brctl 1.31.1

SYNOPSIS

brctl COMMAND [BRIDGE [ARGS]]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Manage ethernet bridges Commands:

show [BRIDGE]...
Show bridges
addbr BRIDGE
Create BRIDGE
delbr BRIDGE
Delete BRIDGE
addif BRIDGE IFACE
Add IFACE to BRIDGE
delif BRIDGE IFACE
Delete IFACE from BRIDGE
stp BRIDGE 1/yes/on|0/no/off
STP on/off
setageing BRIDGE SECONDS
Set ageing time
setfd BRIDGE SECONDS
Set bridge forward delay
sethello BRIDGE SECONDS
Set hello time
setmaxage BRIDGE SECONDS
Set max message age
setbridgeprio BRIDGE PRIO
Set bridge priority
setportprio BRIDGE IFACE PRIO
Set port priority
setpathcost BRIDGE IFACE COST
Set path cost
