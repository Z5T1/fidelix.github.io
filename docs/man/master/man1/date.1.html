DATE(1) User Commands DATE(1)

NAME

date - manual page for date 1.31.1

SYNOPSIS

date [OPTIONS] [+FMT] [TIME]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Display time (using +FMT), or set time

[-s,--set] TIME Set time to TIME -u,--utc Work in UTC (don't convert to local time) -R,--rfc-2822 Output RFC-2822 compliant date string -I[SPEC] Output ISO-8601 compliant date string
SPEC='date' (default) for date only,
'hours', 'minutes', or 'seconds' for date and time to the indicated precision
-r,--reference FILE
Display last modification time of FILE
-d,--date TIME
Display TIME, not 'now'
-D FMT
Use FMT (strptime format) for -d TIME conversion

Recognized TIME formats:

hh:mm[:ss] [YYYY.]MM.DD-hh:mm[:ss] YYYY-MM-DD hh:mm[:ss] [[[[[YY]YY]MM]DD]hh]mm[.ss] 'date TIME' form accepts MMDDhhmm[[YY]YY][.ss] instead
