SHUF(1) User Commands SHUF(1)

NAME

shuf - manual page for shuf 1.31.1

SYNOPSIS

shuf [-e|-i L-H] [-n NUM] [-o FILE] [-z] [FILE|ARG...]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Randomly permute lines

-e
Treat ARGs as lines
-i L-H
Treat numbers L-H as lines
-n NUM
Output at most NUM lines

-o FILE Write to FILE, not standard output

-z
End lines with zero byte, not newline
