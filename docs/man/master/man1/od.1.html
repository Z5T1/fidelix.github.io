SYNOPSIS

od [-abcdfhilovxs] [-t TYPE] [-A RADIX] [-N SIZE] [-j SKIP] [-S MINSTR] [-w WIDTH] [FILE]...

DESCRIPTION

Print FILEs (or stdin) unambiguously, as octal bytes by default
