UNAME(1) User Commands UNAME(1)

NAME

uname - manual page for uname 1.31.1

SYNOPSIS

uname [-amnrspvio]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Print system information

-a
Print all
-m
The machine (hardware) type
-n
Hostname
-r
Kernel release
-s
Kernel name (default)
-p
Processor type
-v
Kernel version
-i
The hardware platform
-o
OS name
