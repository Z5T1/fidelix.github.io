SOFTLIMIT(1) User Commands SOFTLIMIT(1)

NAME

softlimit - manual page for softlimit 1.31.1

SYNOPSIS

softlimit [-a BYTES] [-m BYTES] [-d BYTES] [-s BYTES] [-l BYTES]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.
[-f BYTES] [-c BYTES] [-r BYTES] [-o N] [-p N] [-t N] PROG ARGS

Set soft resource limits, then run PROG

-a BYTES
Limit total size of all segments
-m BYTES
Same as -d BYTES -s BYTES -l BYTES -a BYTES
-d BYTES
Limit data segment
-s BYTES
Limit stack segment
-l BYTES
Limit locked memory size
-o N
Limit number of open files per process
-p N
Limit number of processes per uid

Options controlling file sizes:

-f BYTES
Limit output file sizes
-c BYTES
Limit core file size

Efficiency opts:

-r BYTES
Limit resident set size
-t N
Limit CPU time, process receives a SIGXCPU after N seconds
