NETSTAT(1) User Commands NETSTAT(1)

NAME

netstat - manual page for netstat 1.31.1

SYNOPSIS

netstat [-ral] [-tuwx] [-enWp]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Display networking information

-r
Routing table
-a
All sockets
-l
Listening sockets Else: connected sockets
-t
TCP sockets
-u
UDP sockets
-w
Raw sockets
-x
Unix sockets Else: all socket types
-e
Other/more information
-n
Don't resolve names
-W
Wide display
-p
Show PID/program name for sockets
