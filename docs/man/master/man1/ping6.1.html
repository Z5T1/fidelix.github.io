PING6(1) User Commands PING6(1)

NAME

ping6 - manual page for ping6 1.31.1

SYNOPSIS

ping6 [OPTIONS] HOST

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Send ICMP ECHO_REQUEST packets to network hosts

-c CNT
Send only CNT pings
-s SIZE
Send SIZE data bytes in packets (default 56)
-i SECS
Interval
-A
Ping as soon as reply is recevied
-I IFACE/IP
Source interface or IP address
-q
Quiet, only display output at start and when finished
-p HEXBYTE
Pattern to use for payload
