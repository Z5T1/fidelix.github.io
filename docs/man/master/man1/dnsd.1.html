DNSD(1) User Commands DNSD(1)

NAME

dnsd - manual page for dnsd 1.31.1

SYNOPSIS

dnsd [-dvs] [-c CONFFILE] [-t TTL_SEC] [-p PORT] [-i ADDR]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Small static DNS server daemon

-c FILE Config file

-t SEC
TTL

-p PORT Listen on PORT

-i ADDR Listen on ADDR

-d
Daemonize
-v
Verbose
-s
Send successful replies only. Use this if you want to use /etc/resolv.conf with two nameserver lines:
nameserver DNSD_SERVER
nameserver NORMAL_DNS_SERVER
