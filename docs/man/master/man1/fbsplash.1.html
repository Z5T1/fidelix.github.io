FBSPLASH(1) User Commands FBSPLASH(1)

NAME

fbsplash - manual page for fbsplash 1.31.1

SYNOPSIS

fbsplash -s IMGFILE [-c] [-d DEV] [-i INIFILE] [-f CMD]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.
-s
Image
-c
Hide cursor
-d
Framebuffer device (default /dev/fb0)
-i
Config file (var=value): BAR_LEFT,BAR_TOP,BAR_WIDTH,BAR_HEIGHT BAR_R,BAR_G,BAR_B,IMG_LEFT,IMG_TOP
-f
Control pipe (else exit after drawing image) commands: 'NN' (% for progress bar) or 'exit'
