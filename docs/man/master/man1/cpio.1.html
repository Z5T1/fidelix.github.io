CPIO(1) User Commands CPIO(1)

NAME

cpio - manual page for cpio 1.31.1

SYNOPSIS

cpio [-dmvu] [-F FILE] [-R USER[:GRP]] [-H newc] [-tio] [-p DIR] [EXTR_FILE]...

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Extract (-i) or list (-t) files from a cpio archive, or take file list from stdin and create an archive (-o) or copy files (-p)

Main operation mode:

-t
List
-i
Extract EXTR_FILEs (or all)
-o
Create (requires -H newc)
-p DIR
Copy files to DIR

OPTIONS

-H newc Archive format

-d
Make leading directories
-m
Preserve mtime
-v
Verbose
-u
Overwrite

-F FILE Input (-t,-i,-p) or output (-o) file

-R USER[:GRP]
Set owner of created files
-L
Dereference symlinks
-0
Input is separated by NULs
