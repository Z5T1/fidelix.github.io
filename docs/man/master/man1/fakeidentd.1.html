FAKEIDENTD(1) User Commands FAKEIDENTD(1)

NAME

fakeidentd - manual page for fakeidentd 1.31.1

SYNOPSIS

fakeidentd [-fiw] [-b ADDR] [STRING]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Provide fake ident (auth) service

-f
Run in foreground
-i
Inetd mode
-w
Inetd 'wait' mode

-b ADDR Bind to specified address

STRING
Ident answer string (default: nobody)
