NETCAT(1) User Commands NETCAT(1)

NAME

netcat - manual page for netcat 1.31.1

SYNOPSIS

netcat [OPTIONS] HOST PORT - connect

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

nc [OPTIONS] -l -p PORT [HOST] [PORT] - listen

-e PROG Run PROG after connect (must be last)

-l
Listen mode, for inbound connects
-lk
With -e, provides persistent server

-p PORT Local port

-s ADDR Local address

-w SEC
Timeout for connects and final net reads
-i SEC
Delay interval for lines sent
-n
Don't do DNS resolution
-u
UDP mode
-v
Verbose

-o FILE Hex dump traffic

-z
Zero-I/O mode (scanning)
