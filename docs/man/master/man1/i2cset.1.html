I2CSET(1) User Commands I2CSET(1)

NAME

i2cset - manual page for i2cset 1.31.1

SYNOPSIS

i2cset [-fy] [-m MASK] BUS CHIP-ADDRESS DATA-ADDRESS [VALUE] ... [MODE]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Set I2C registers

I2CBUS
I2C bus number
ADDRESS 0x03-0x77

MODE is:

c
Byte, no value
b
Byte data (default)
w
Word data
i
I2C block data
s
SMBus block data
Append p for SMBus PEC
-f
Force access
-y
Disable interactive mode
-r
Read back and compare the result

-m MASK Mask specifying which bits to write
