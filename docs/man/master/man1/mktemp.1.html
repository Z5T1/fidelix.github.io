MKTEMP(1) User Commands MKTEMP(1)

NAME

mktemp - manual page for mktemp 1.31.1

SYNOPSIS

mktemp [-dt] [-p DIR] [TEMPLATE]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Create a temporary file with name based on TEMPLATE and print its name. TEMPLATE must end with XXXXXX (e.g. [/dir/]nameXXXXXX). Without TEMPLATE, -t tmp.XXXXXX is assumed.

-d
Make directory, not file
-q
Fail silently on errors
-t
Prepend base directory name to TEMPLATE
-p DIR
Use DIR as a base directory (implies -t)
-u
Do not create anything; print a name

Base directory is: -p DIR, else $TMPDIR, else /tmp
