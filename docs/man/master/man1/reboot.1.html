REBOOT(1) User Commands REBOOT(1)

NAME

reboot - manual page for reboot 1.31.1

SYNOPSIS

reboot [-d DELAY] [-n] [-f]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Reboot the system

-d SEC
Delay interval
-n
Do not sync
-f
Force (don't go through init)
