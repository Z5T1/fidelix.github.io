SLATTACH(1) User Commands SLATTACH(1)

NAME

slattach - manual page for slattach 1.31.1

SYNOPSIS

slattach [-ehmLF] [-c SCRIPT] [-s BAUD] [-p PROTOCOL] SERIAL_DEVICE

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Configure serial line as SLIP network interface

-p PROT Protocol: slip, cslip (default), slip6, clisp6, adaptive

-s BAUD Line speed

-e
Exit after initialization
-h
Exit if carrier is lost (else never exits)

-c PROG Run PROG on carrier loss

-m
Do NOT set raw 8bit mode
-L
Enable 3-wire operation
-F
Disable RTS/CTS flow control
