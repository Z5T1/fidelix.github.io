BLOCKDEV(1) User Commands BLOCKDEV(1)

NAME

blockdev - manual page for blockdev 1.31.1

SYNOPSIS

blockdev OPTION BLOCKDEV

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.
--setro
Set ro
--setrw
Set rw
--getro
Get ro
--getss
Get sector size
--getbsz
Get block size
--setbsz BYTES
Set block size
--getsz
Get device size in 512-byte sectors
--getsize64
Get device size in bytes
--flushbufs
Flush buffers
--rereadpt
Reread partition table
