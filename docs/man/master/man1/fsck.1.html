FSCK(1) User Commands FSCK(1)

NAME

fsck - manual page for fsck 1.31.1

SYNOPSIS

fsck [-ANPRTV] [-t FSTYPE] [FS_OPTS] [BLOCKDEV]...

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Check and repair filesystems

-A
Walk /etc/fstab and check all filesystems
-N
Don't execute, just show what would be done
-P
With -A, check filesystems in parallel
-R
With -A, skip the root filesystem
-T
Don't show title on startup
-V
Verbose

-t TYPE List of filesystem types to check
