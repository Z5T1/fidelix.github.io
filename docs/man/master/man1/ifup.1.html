IFUP(1) User Commands IFUP(1)

NAME

ifup - manual page for ifup 1.31.1

SYNOPSIS

ifup [-anmvf] [-i FILE] IFACE...

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.
-a
Configure all interfaces

-i FILE Use FILE instead of /etc/network/interfaces

-n
Print out what would happen, but don't do it (note: doesn't disable mappings)
-m
Don't run any mappings
-v
Print out what would happen before doing it
-f
Force configuration
