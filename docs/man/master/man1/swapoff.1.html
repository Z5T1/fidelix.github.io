SWAPOFF(1) User Commands SWAPOFF(1)

NAME

swapoff - manual page for swapoff 1.31.1

SYNOPSIS

swapoff [-a] [DEVICE]

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Stop swapping on DEVICE

-a
Stop swapping on all swap devices
