TAIL(1) User Commands TAIL(1)

NAME

tail - manual page for tail 1.31.1

SYNOPSIS

tail [OPTIONS] [FILE]...

DESCRIPTION

BusyBox v1.31.1 (2020-04-30 13:38:01 EDT) multi-call binary.

Print last 10 lines of each FILE (or stdin) to stdout. With more than one FILE, precede each with a filename header.

-f
Print data as file grows
-c [+]N[kbm]
Print last N bytes
-n N[kbm]
Print last N lines
-n +N[kbm]
Start on Nth line and print the rest
-q
Never print headers
-s SECONDS
Wait SECONDS between reads with -f
-v
Always print headers
-F
Same as -f, but keep retrying

N may be suffixed by k (x1024), b (x512), or m (x1024^2).
